How to install BabelAS2 to send messages to the mendelson AS2 test server

Patrice, 2009-11-09

1. Unzip BabelAS2.V0.01.zip in a <folder>.
[Install the certificate and private key of the sender (mycompanyAS2)]
2. Double-click on "<folder>\BabelAS2\Cert\key1.pfx"
3. Click "Next"
4. Click "Next"
5. Type the password "test", leave the two check boxes and click "Next"
6. Select the radio button "Place all certificates in the following store", click the "Browse..." button, select the "Personal" certificate store, click "OK", and click "Next"
7. Click "Finish".
[Install the certificate of the receiver (mendelsontestAS2)]
8. Right-click on "<folder>\BabelAS2\Cert\key2.cer" and select "Install Certificate"
9. Click "Next"
10. Select the radio button "Place all certificates in the following store", click the "Browse..." button, select the "Personal" certificate store, click "OK", and click "Next"
11. Click "Finish".
[Send the test EDIFACT message to the mendelson AS2 test server]
12. Double-click on "<folder>\BabelAS2\BabelAS2.Send.vbs"
